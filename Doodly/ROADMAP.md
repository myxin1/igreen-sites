# ROADMAP — doodly.online (Reconstrução Completa)

> Site afiliado multilíngue para o software Doodly
> Idiomas: PT 🇧🇷 | EN 🇺🇸 | ES 🇪🇸 | NL 🇳🇱
> Estrutura: SILO temático por idioma
> Objetivo: Rankear no Google e converter via link afiliado

---

## FASE 1 — Fundação Técnica (Semana 1–2)

### WordPress / CMS
- [ ] Instalar WordPress limpo
- [ ] Instalar tema leve (GeneratePress ou Astra — sem Elementor pesado)
- [ ] Instalar Rank Math SEO (gratuito)
- [ ] Instalar plugin de cache (WP Rocket ou LiteSpeed)
- [ ] Configurar HTTPS + redirecionar HTTP → HTTPS
- [ ] Ativar lazy load para imagens
- [ ] Configurar CDN (Cloudflare gratuito)

### Rank Math — Configurações críticas
- [ ] Ativar módulo hreflang
- [ ] Ativar módulo Schema
- [ ] Configurar sitemap separado por idioma
- [ ] Configurar Open Graph (para compartilhamento social)

### Estrutura de URLs por idioma
```
doodly.online/                    → Homepage EN (x-default)
doodly.online/pt/                 → Hub PT
doodly.online/es/                 → Hub ES
doodly.online/nl/                 → Hub NL
```

### Google
- [ ] Criar/verificar Google Search Console
- [ ] Enviar sitemap para todos os idiomas
- [ ] Criar Google Analytics 4
- [ ] Conectar GSC ao GA4
