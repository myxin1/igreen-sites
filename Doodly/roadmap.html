REST API — Acesso Remoto (Claude)
Como configurar (sem plugin):
1. WordPress Admin → Usuários → seu perfil → role abaixo até "Application Passwords"
2. Nome: Claude Access → clicar Add New Application Password
3. Copiar o token gerado (aparece só uma vez)
4. Passar para o Claude no formato: usuario:token em Base64 no header Authorization

Endpoints que serão usados:
GET /wp-json/wp/v2/posts // listar artigos
POST /wp-json/wp/v2/posts // criar artigo
PUT /wp-json/wp/v2/posts/{id}// editar artigo
GET /wp-json/wp/v2/pages // listar páginas
POST /wp-json/rankmath/v1/updateMeta// SEO via Rank Math API
